Cucumber JVM is the Java implementation of Cucumber — a tool for running automated tests written in plain language. Because tests are written in Gherkin (Given/When/Then), they can be read and understood by everyone on your team: developers, QA engineers, product managers, and business stakeholders alike.

## What is Cucumber JVM?

Cucumber JVM bridges the gap between business requirements and automated tests. You write feature files in the Gherkin language that describe your application's behavior in plain English. Cucumber then maps each step in those files to Java methods — called step definitions — that perform the actual test actions.

```gherkin src/test/resources/com/example/calculator.feature
Feature: Calculator
  Scenario: Adding two numbers
    Given I have a calculator
    When I add 2 and 3
    Then the result should be 5
```

```java src/test/java/com/example/CalculatorSteps.java
package com.example;

import io.cucumber.java.en.Given;
import io.cucumber.java.en.When;
import io.cucumber.java.en.Then;
import static org.junit.jupiter.api.Assertions.assertEquals;

public class CalculatorSteps {

    private Calculator calculator;
    private int result;

    @Given("I have a calculator")
    public void iHaveACalculator() {
        calculator = new Calculator();
    }

    @When("I add {int} and {int}")
    public void iAddAndNumbers(int a, int b) {
        result = calculator.add(a, b);
    }

    @Then("the result should be {int}")
    public void theResultShouldBe(int expected) {
        assertEquals(expected, result);
    }
}
```

## Ecosystem modules

Cucumber JVM is published as a set of focused Maven artifacts. You compose what you need:

| Module | Purpose |
|--------|---------|
| `cucumber-java` | Core annotations and step definition support |
| `cucumber-junit-platform-engine` | JUnit 5 test engine (recommended) |
| `cucumber-junit` | JUnit 4 runner (maintenance mode) |
| `cucumber-testng` | TestNG integration |
| `cucumber-spring` | Spring / Spring Boot dependency injection |
| `cucumber-guice` | Google Guice dependency injection |
| `cucumber-picocontainer` | PicoContainer dependency injection |
| `cucumber-cdi2` | CDI 2 / Jakarta CDI dependency injection |
| `datatable` | DataTable conversion utilities |
| `cucumber-bom` | Bill of Materials for version management |
